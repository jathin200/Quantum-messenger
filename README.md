# 🔐 Quantum Messenger

## 📌 Overview
Quantum Messenger is a secure peer-to-peer offline messaging application designed for disaster recovery and emergency communication scenarios. The application enables communication without internet connectivity using nearby device connections and adaptive cryptographic security.

## 🚀 Features
- Offline peer-to-peer messaging
- End-to-end encrypted communication
- Nearby Connections API integration
- Dynamic Security Policy Engine
- Post-Quantum Cryptography support
- Contact management system
- Cross-platform Flutter application

## 🛠️ Technologies Used
### Frontend

### Security

### Storage

## 🏗️ Architecture
The system follows a Software Defined Security (SDS) architecture:
- Control Plane
- Data Plane
- Dynamic Cryptographic Policy Engine (DCPE)
- Context-Aware Security Selection

## 🔒 Security Features
- Dynamic cryptographic algorithm selection
- Context-based security policies
- Trust-aware communication
- Energy-efficient encryption
